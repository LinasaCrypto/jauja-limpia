# JAUJA LIMPIA

## Descripción

JAUJA LIMPIA es una plataforma web de participación ciudadana diseñada para reportar problemas relacionados con la limpieza urbana, acumulación de basura, contaminación y falta de servicios de limpieza en la ciudad de Jauja.

## Características

- **Reportes Ciudadanos**: Permite a los ciudadanos reportar problemas de limpieza urbana
- **Formulario de Reporte**: Recopila información detallada sobre incidentes
- **Sistema de Ubicación**: Permite obtener la ubicación GPS de los reportes
- **Visualización de Estadísticas**: Muestra métricas de reportes recibidos y actividad ciudadana
- **Interfaz Moderna**: Diseño responsive y atractivo inspirado en gobierno digital y smart city

## Tecnologías Utilizadas

- HTML5
- CSS3 (estilos personalizados)
- JavaScript Vanilla (sin frameworks)
